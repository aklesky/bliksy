## Usage for Prettier Preset

```json
{
    "prettier": "@aklesky/prettier-config"
}
```

## Preset

> Prettier Configuration enabled by default

```json
{
    "printWidth": 120,
    "semi": false,
    "singleQuote": true,
    "tabWidth": 4,
    "trailingComma": "all",
    "arrowParens": "avoid",
}
```

### Supported Prettier versions

| Name    | Version | Supported          |
| ------- | ------- | ------------------ |
| Prettier    | >= 2.8.3 | :white_check_mark: |
